# IRR WordPress Theme

Tema custom `irr-theme` para el sitio de Interest Rate Research.

## Requisitos

- WordPress local (por ejemplo LocalWP)
- Tema activo: `irr-theme`

## Estructura

- `functions.php`: setup del tema y hooks
- `page-get-started.php`: flujo de formulario y calendario
- `assets/js/main.js`: interacciones frontend
- `style.css`: estilos globales y de plantillas

## Git

Este repositorio esta pensado para versionar solo el tema, no todo WordPress.
